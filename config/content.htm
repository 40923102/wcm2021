This is https://github.com/mdecourse/cmstemplate

目前的 cmsimde 在編輯器下方新增一個 csave 按鈕, 意即 collabrative save, 當動態網際管理系統在多人同時維護登入維護網頁內容時, 編輯各頁面時段, 該頁面可能已經改版, 因此使用 csave 按鈕存檔時, 會導入當下最新的該頁面內容, 並試圖與編輯中的頁面內容進行合併.

使用 csave 按鈕存檔無法刪除頁面資料.

使用 Edit All 模式, 無法使用 csave 按鈕（尚未測試).

為什麼會在機械設計工程系開設網際內容管理課程?

 

  • 利用瀏覽器作為前端, 並將數位內容有系統地送到雲端伺服器中進行管理與運用的架構即稱為網際內容管理.
  • 從機械設計專業的角度看待網際內容管理.
  • 機械設計工程師應該從 CAD 出發透過零組件系統建立, 模擬與控制, 並在流程中利用 CAE 進行設計功能確認後, 將所有持續改進的內容透過分散式版次與網際管理系統進行保存與數位資料再利用.
  • 機械系的巔峰 (其他專業科系是否也適用?)https://www.ptt.cc/bbs/Tech_Job/M.1588362728.A.14F.html
